init
=====

1. create a folder
2. git init
3. create a file
4. git add file
5. git commit
6. create a repo in git
7. git config --global user.username pranavatgithub
8. git remote add origin https://github.com/pranavatgithub/FCC_TEST.git
9. git push origin master // origin is just a name
10.git pull origin develop 
11. git clone https://github.com/pranavatgithub/FCC_TEST.git // make a copy
12.git remote add upstream https://github.com/pranavatgithub/FCC_TEST.git // connecting with a upstream repo
